# Task-1
# AI Website Copy Generator for Local Business  
Task 1 – Prompt Engineering Internship (Future Interns)

---

## 📌 Project Overview

This project demonstrates a structured prompt engineering workflow to generate high-converting website copy for a local business using AI tools.

The objective was to design repeatable prompts that produce:

- Homepage copy
- Service descriptions
- Persuasive Call-to-Action (CTA) content

All outputs were generated using structured prompts and documented for verification.

---

## 🏢 Selected Business

**Business Name:** GlowNest Salon  
**Location:** Anna Nagar, Chennai  
**Target Audience:** Women (18–40)  
**Brand Tone:** Premium + Friendly + Modern  
**Primary Goal:** Increase Bookings  

---

## 🎯 Problem Statement

Many local businesses struggle with:

- Unclear value proposition
- Generic website copy
- Weak call-to-actions
- Low conversion rates

Professional copywriters are expensive. This project demonstrates how structured prompts can generate conversion-focused website content efficiently.

---

## 🧠 Prompt Engineering Approach

The workflow followed:

1. Defined business inputs (name, location, audience, tone, goal)
2. Designed structured prompts for:
   - Homepage
   - Services
   - CTA section
3. Generated AI outputs
4. Refined content to improve:
   - Emotional appeal
   - Local relevance (Anna Nagar, Chennai)
   - Conversion focus
5. Compiled final polished website copy
